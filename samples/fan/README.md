# 泛音采样(金玉之声音源层)

把每根弦的**七徽泛音**录音放进本目录,命名为弦号:

```
samples/fan/1.wav   ← 一弦七徽泛音(约 C3, 130.8Hz)
samples/fan/2.wav   ← 二弦(约 D3)
...
samples/fan/7.wav   ← 七弦(约 D4, 293.7Hz)
```

- 支持 .wav / .m4a / .mp3(手机语音备忘录的 m4a 可直接用)
- 每条 2–5 秒:轻触七徽、拨响、抬指,让余音自然收尾;安静环境,离琴 30–50cm
- 引擎自动:去首部静音、峰值归一、按目标徽位变速播放(其他徽的泛音由七徽样本移调)
- 缺哪根弦就自动回退合成泛音,不会坏
- 若使用第三方录音,许可需允许商用(CC0/CC-BY;CC-BY 需在应用中署名)
