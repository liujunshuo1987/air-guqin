# 无弦琴 · 凌空古琴

> 「但识琴中趣,何劳弦上声」 —— 陶渊明

单摄像头的"无弦之琴":七弦十三徽布于镜前空间,手势凌空演奏,Karplus-Strong 物理合成发声。无实体、无穿戴,一个浏览器页面即是一张琴。

![](https://img.shields.io/badge/tech-MediaPipe%20%2B%20AudioWorklet%20%2B%20Canvas-8a7a55)

## 快速开始

```bash
python3 -m http.server 8093
# Chrome/Edge 打开 http://localhost:8093,点「入席抚琴」,允许摄像头
```

需联网加载 MediaPipe 手部模型(首次约 8MB)。无摄像头时可用鼠标点弦/拖动试音,键盘 1–7 弹散音。

## 演奏法(立面·悬指,默认模式)

琴身立起如镜:弦横陈,岳山在右,徽位向左。两手以站位分工——**靠岳山(右)侧的手司拨弦,另一只司按弦**。

| 动作 | 效果 |
|---|---|
| 拨弦手 伸指上下划过琴弦 | 拨弦(力度随手速);顺势下扫七弦=滚拂 |
| 拨弦手 握拳移动(指尖收回掌心) | 换弦静默,过弦不响 |
| 按弦手 左右悬指 | 定徽位(不必触弦);移动=走手音,小幅抖=吟猱 |
| 按弦手 拇食捏合 | 泛音(自动吸附徽位,按真实谐波次数发声) |
| 按弦手 垂出弦带 | 散音 |
| 双手左右平举静止约一秒 | **定琴**:琴落于双手之间,大小随距离自适应,定成自动上锁(右上🔒可解) |

定琴之后,弦的位置由你的身体记忆决定——可以不看屏幕盲弹;悬指经过徽位有轻微提示音助校准。

## 跟谱模式(减字谱识读)

右上角「琴谱」选曲即开。减字谱本是指位谱而非节奏谱,故采用**自走式跟谱**:弹对(弦对+徽位在容差内+散/按/泛声型对)即前进,不卡节奏;弹错给方向提示。界面呈现圆形减字牌+队列+逐字唱词,右上角以魏碑体呈现减字谱近似原貌,目标弦发光、目标徽位有幽灵参考线。

曲库 11 首按难度递进:入门散按泛 → 小星星 → 两只老虎 → 仙翁操(开指)→ 茉莉花 → 沧海一声笑 → 送别 → 流水(泛音)→ 酒狂 → 阳关 → **神人畅**(72 音长篇:泛音开指→低音吟诵→按音升腾→泛音归羽)。

> 命名约定:民歌/流行曲标「移植」(旋律照原曲);传世古曲标「意临」(非逐音打谱,取其调式织体气韵的致意改编)。欢迎按打谱资料把真谱录入 `SCORES`。

## 录制

「⏺ 录制」(或按 R)输出微信可直接播放的 **MP4**(H.264+AAC)。「含提示」开=全界面录制(标签页屏幕捕获,含全部 UI);关=只录琴面并隐去提示,出干净演奏视频。音频直取合成器输出,无环境噪音。

## 技术要点

- **手部追踪**:MediaPipe HandLandmarker(双手 21 关键点,GPU);"触弦态"手指 = 径向伸直 **或** 指尖深度前探(勾/撮爪形在二维投影中与拳无异,靠 z 分量区分)
- **发声**:AudioWorklet 内 Karplus-Strong 弦合成 + 琴体共鸣滤波;正调定弦 C2–D3,十三徽按真实弦长比,泛音按徽位谐波次数;走手音=延迟线连续变调
- **距离无关**:所有速度/力度以演奏者手掌成像大小归一化,远近皆同一手感;HUD 有距离教练与力度百分比
- **设备无关**:只在摄像头出新帧时推理,速度按真实帧距归一——15/30/60fps 摄像头手感一致且省电;多摄像头可下拉切换(记住选择);支持触屏点弦/拖动;演奏时申请屏幕常亮
- **离线 PWA**:MediaPipe 模型与 wasm 已本地化(`vendor/`,本地缺失自动退回 CDN);Service Worker 预缓存应用壳与模型,断网可用;可"添加到主屏幕"独立运行(建议横屏)
- **单文件逻辑**:全部代码在 [index.html](index.html),无构建步骤

## 已知约定

- 摄像头信号源若已预翻转(手机 webcam/虚拟摄像头),左右手会认反——切换右上角「镜像」即可(选择会记住)
- 浏览器后台标签页会挂起渲染循环,演奏/录制时请保持页面可见
- **手机浏览器访问必须 HTTPS**(getUserMedia 安全要求,localhost 例外只对本机有效):可用 GitHub Pages/静态托管,或 cloudflared/ngrok 临时隧道
- iOS 不支持标签页屏幕捕获,「含提示」全界面录制会自动退回画布模式
